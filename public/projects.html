Studio Connect

Studio Connect is an engaging chatroom web app designed for music producers seeking real-time collaboration and knowledge sharing. Built with MongoDB, Express, React, and Node (MERN stack), it features secure user authentication (complete with signup and login screens), Mongoose schemas for robust data validation, and dedicated chat rooms that producers can join or leave at will. A real-time message typing indicator enhances communication flow.

Currency Flow

An intuitive currency conversion app built with React Native, perfect for travelers, online shoppers, or anyone needing up-to-date exchange rates. By integrating with a freecurrencyapi, it retrieves real-time conversion data and calculates precise amounts between a chosen base currency and destination currency.
